# 🏏 Pro Cricket Trials FYP

## 📌 Description
This Final Year Project is an AI-powered Cricket Trials Management System developed to automate player evaluation using video analysis and smart algorithms.

## 🚀 Features (from poster)
- Player Registration  
- Performance Tracking  
- Dashboard Visualization  
- AI-based Player Analysis  
- Report Generation (PDF)

## 🛠️ Tools Used
- Python
- HTML, CSS, JavaScript (if used in website)  
- MySQL (if database used)

## 🎓 Project Purpose
To modernize cricket trials by using Artificial Intelligence for fair and efficient player evaluation.
